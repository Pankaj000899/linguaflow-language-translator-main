# LinguaFlow — Language Translator Web App

LinguaFlow is a modern and responsive language translation web application built using HTML, CSS, and JavaScript.

It allows users to translate text between multiple languages with a clean glassmorphism UI and advanced features like voice input, text-to-speech, translation history, dark/light mode, PDF/TXT download, and responsive design.

## Features

🌍 Multi-language translation
🎤 Voice input support
🔊 Text-to-speech output
📜 Translation history
🌙 Dark/Light mode
📄 Download translation as TXT/PDF
📋 Copy translated text
⚡ Auto translate support
📱 Fully responsive modern UI
⭐ Favourite translations
🔎 Search translation history

## Tech Stack

* HTML5
* CSS3
* JavaScript
* MyMemory Translation API

## How to Run

1. Download or clone the repository
2. Open the project folder
3. Run `index.html` in browser
